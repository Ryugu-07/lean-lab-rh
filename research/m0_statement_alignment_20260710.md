## Primary Sources

1. A. Beurling, *A Closure Problem Related to the Riemann Zeta-Function* (1955),
   <https://doi.org/10.1073/pnas.41.5.312>.
   The unit-interval linear manifold uses finite sums with `0 < theta < 1` and the coefficient
   condition `sum c_k * theta_k = 0`.
2. Balazard and Saias, *The Nyman-Beurling Equivalent Form for the Riemann Hypothesis* (1998),
   <https://www.esi.ac.at/preprints/esi623.pdf>.
   Formula (2) uses modified generators `rho(theta / x) - theta * rho(1 / x)`, which encode the
   same zero-moment restriction. Section 5 also records the related full-line raw-generator form.
3. L. Baez-Duarte, *A strengthening of the Nyman-Beurling criterion for the Riemann Hypothesis*
   (2002), <https://arxiv.org/abs/math/0202141>.
   Theorem 1.1 uses `H = L2(0,infinity)`, target `chi_(0,1]`, and the span generated by
   `rho(1 / (n*x))` for positive natural `n`.

## Lean-Checked Falsification

The project now proves:

```lean
theorem nymanBeurlingConcreteApprox_unconditional :
    nymanBeurlingConcreteApprox
```

The witness uses coefficients one at parameters `1` and `-1`. Outside the countable set
`{z^(-1) | z : Int}`, the fractional part of `1/x` is nonzero, so `Int.fract_neg` gives

```text
fract (1/x) + fract (-1/x) = 1.
```

The squared approximation error is therefore zero almost everywhere on `(0,1)`. This proof has no
RH premise.
