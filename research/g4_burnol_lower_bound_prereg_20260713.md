The fixed published target is Burnol's theorem for

```text
B_lambda = span_C { t |-> rho(theta / t) : lambda <= theta <= 1 }
D(lambda) = dist(chi_(0,1], B_lambda),       0 < lambda < 1,
```

namely

```text
liminf_(lambda -> 0)
  D(lambda) * sqrt(log(1/lambda))
    >= sqrt(sum_rho m_rho^2 / |rho|^2).
```

The sum ranges over nontrivial zeta zeros and `m_rho` is the zero multiplicity. For the project's
natural space generated by `rho(1/(n*t))`, `1 <= n <= N`, inclusion in `B_(1/N)` gives the
published lower-bound consequence for the natural distance `d_N`:

```text
D(1/N) <= d_N,
```

so the squared natural distance cannot decay faster than order `1/log N` when the right-hand
constant is positive.

## Primary Source

Jean-Francois Burnol, *A lower bound in an approximation problem involving the zeros of the
Riemann zeta function*, Advances in Mathematics 170 (2002), 56-70, arXiv `math/0103058v2`:

- <https://arxiv.org/abs/math/0103058>
- source submission dated 2001-03-20;
- downloaded source SHA-256:

The source first quotes the Baez-Duarte-Balazard-Landreau-Saias lower bound with each zero counted
once, then strengthens the constant to squared multiplicities. It explicitly notes that the
inequality is trivial if RH fails and proves the nontrivial case under RH.
